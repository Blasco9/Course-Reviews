## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- Ruby: 2.7.0
- Rails: 6.0.3.2
- Postgres: >=9.5

### Setup

Instal gems with:

`bundle install`

Setup database with:

`rails db:create
   rails db:migrate`

### Usage

Start server with:

`rails server`

Open http://localhost:3000/ in your browser.

### Run tests

`rpsec --format documentation`
